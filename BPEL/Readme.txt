Add Apache ODE 1.x server in Windows -> Show View -> Servers.3.	Right click on the server and click on Add and Remove and add TempConversion4.	Start the server5.	Right click on WS_InvocationArtifacts.wsdl file and Web Services -> Test with Web Service Explorer6.	Enter the required inputs and click on go to get the response from web service.
