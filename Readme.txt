To run the project,
1) Open Eclipse and create a new java project.
2) Copy and replace api.txt, src, bin, Index, Categorized inside project directory.
3) Right click on project in IDE, click properties and select Build path.
4) Add the following external jars,
com.google.guava_1.6.0.jarejml-0.23.jarguava-15.0-rc1-site.jarjsoup-1.10.2-javadoc.jarjsoup-1.10.2-sources.jarjsoup-1.10.2.jarlucene-analyzers-common-4.0.0.jarlucene-core-4.0.0.jarlucene-queryparser-4.0.0.jarslf4j-api-1.7.12-sources.jarslf4j-api.jarstanford-parser-3.7.0-javadoc.jarstanford-parser-3.7.0-models.jarstanford-parser-3.7.0-sources.jarstanford-parser.jarstanford-postagger-3.7.0-javadoc.jarstanford-postagger-3.7.0-sources.jarstanford-postagger-3.7.0.jarstanford-postagger.jar

5) Right click on UserIntentsMining.java and run as Java application.
